・うまく動かない方へ・

ブラウザのカメラ使用許可を拒否にしてしまった

【Android】

Chromeを利用している前提にてお話します。

● まず試す方法

  1. アドレスバー左側の 🔒（鍵マーク） または ⓘ をタップ
  2. 「権限」または「サイトの設定」をタップ
  3. 「カメラ」を「許可」に変更
  4. ページを再読み込み（更新）

● 上記で見つからない場合

  1. Chrome 右上の ︙（メニュー） をタップ
  2. 「設定」→「サイトの設定」→「カメラ」
  3. 「ブロック中」の一覧からこのサイトを選び、「許可」に変更
  4. ページを再読み込み

● それでも使えない場合（端末の設定から）

  1. スマートフォンの「設定」アプリを開く
  2. 「アプリ」→「Chrome」
  3. 「権限」→「カメラ」を「許可」に変更
  4. ブラウザに戻り、ページを再読み込み

【iPhone】

iPhone Safariの場合

● まず試す方法

  1. アドレスバーの「ぁあ」または「AA」をタップ
  2. 「Webサイトの設定」をタップ
  3. 「カメラ」を「許可」または「確認」に変更
  4. ページを再読み込み

● 上記で見つからない場合（Safariの設定から）

  1. 「設定」アプリを開く
  2. 「アプリ」→「Safari」をタップ
  3. 下の方の「カメラ」を「確認」または「許可」に変更
  4. Safariに戻り、ページを再読み込み

● それでも使えない場合（端末の設定から）

  1. 「設定」アプリを開く
  2. 「アプリ」→「Safari」をタップ
  3. 「カメラ」の項目をオンに変更
  4. Safariに戻り、ページを再読み込み

※ スクリーンタイムで制限がかかっている場合は「設定」→「スクリーンタイム」→「コンテンツとプライバシーの制限」→「カメラ」も併せてご確認ください。

iPhone Chromeの場合

● まず試す方法

  1. アドレスバー左側の鍵マークをタップ
  2. 「カメラ」を「許可」に変更
  3. ページを再読み込み

● 上記で見つからない場合（Chromeアプリの権限を確認）

  1. 「設定」アプリを開く
  2. 「アプリ」→「Chrome」をタップ
  3. 「カメラ」の項目をオンに変更
  4. Chromeに戻り、ページを再読み込み

● それでも使えない場合（Safariの設定も確認）

  1. 「設定」アプリを開く
  2. 「アプリ」→「Safari」をタップ
  3. 下の方の「カメラ」を「確認」または「許可」に変更
  4. Chromeに戻り、ページを再読み込み

※ iPhone版Chromeは内部的にSafariと同じ仕組みで動いているため、Safari側の設定も影響することがあります。

● スクリーンタイムで制限されている場合

  1. 「設定」アプリを開く
  2. 「スクリーンタイム」をタップ
  3. 「コンテンツとプライバシーの制限」→「カメラ」を「許可」に変更
  4. Chromeに戻り、ページを再読み込み

【それ以外のブラウザ】

申し訳ありませんが、本webアプリの対応ブラウザがスマートフォンのSafari・Chromeだけとなります。それ以外のブラウザの対応策は個別にお探しください。
